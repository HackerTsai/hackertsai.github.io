Message Encryptor - 把訊息變成亂碼

開發中

Message Encryptor 是一款能將訊息變成亂碼的應用程式，

想寄 Email 給朋友？先約定好金鑰，再將訊息用 Message Encryptor 加密，傳給他，他再用 Message Encryptor 解密，

就算信被看到，訊息也不會外洩！！！

原始碼庫：https://github.com/HackerTsai/message-encryptor

功能

  • 加密純文字訊息
  • 加密純文字檔案（*.txt）
